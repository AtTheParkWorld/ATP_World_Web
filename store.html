🔗 Connect Shopify

Your ATP store loads products live from Shopify. To activate it, you need a Storefront API access token from your Shopify admin. Here's how:

  1. Go to your Shopify admin: admin.shopify.com/store/atp-store-7903
  2. Click Settings → Apps and sales channels → Develop apps
  3. Click Create an app → name it "ATP Website"
  4. Click Configure Storefront API scopes
  5. Enable: unauthenticated_read_product_listings and unauthenticated_write_checkouts
  6. Click Save → Install app → Reveal token
  7. Copy the token and paste it into this file: find SHOPIFY_STOREFRONT_TOKEN below and replace YOUR_TOKEN_HERE
// In this file, find and replace:
var SHOPIFY_STOREFRONT_TOKEN = 'YOUR_TOKEN_HERE';
// with your real token:
var SHOPIFY_STOREFRONT_TOKEN = 'shpat_xxxxxxxxxxxx';
